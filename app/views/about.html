A little weather app! - Version 4: Release 2

Background:

WeatherTop Inc. produce a modular weather station called the WeatherTop 1000. This lowcost device takes a set of meteorological readings and displays these on a simple LCD display. This is the Web application for the device.

As WeatherTop Inc. are new to the software industry they are looking to ameliorate any risk by insisting on a working version of the application, even if not all of the features they requested are implemented. They have structured an incremental payment schedule accordingly, with 5 versions as follows: Version Package Label Payment

What does this version include?

  • Weather Code (number in range 100-800), integer
  • Temp (C) decimal
  • Wind Speed (kM/hr) decimal
  • Pressure (hPa) Number
  • Wind Direction
  • User Accounts
  • Location of Station
  • Current Weather Icon
  • Max/Min Values
The dashboard for the application presents, for each station, the latest weather at that station including Wind Compass and Wind Chill. As user you can add & delete stations and weather readings. Note the user have to login or sign up to access their weather data.
